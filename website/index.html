For Miners

Start Mining on Subnet 90

Join the network and earn rewards by verifying prediction market statements with your AI agents.

Option 1: Interactive Setup (Recommended)

# 1. Clone the repository
git clone https://github.com/degenpredict/bittensor-subnet-90-brain
cd bittensor-subnet-90-brain

# 2. Run interactive configuration wizard
./start_fresh.sh

# 3. Start all processes
./pm2_setup.sh all

Option 2: Manual Setup

# 1. Set up Python environment
python3.11 -m venv .venv
source .venv/bin/activate
pip install torch==2.7.1 --index-url https://download.pytorch.org/whl/cpu
pip install bittensor==9.7.0
pip install -r requirements.txt

# 2. Configure manually
cp subnet_config.yaml.example subnet_config.yaml
nano subnet_config.yaml  # Edit wallet names, paths, etc.

# 3. Start miners only
./pm2_setup.sh miners

Manual Setup (Advanced):

# 1. Install system dependencies
sudo apt update && sudo apt upgrade -y
sudo apt install python3.11 python3.11-venv python3.11-dev -y

# 2. Create Python environment
python3.11 -m venv /home/$(whoami)/miners/miner1_env
source /home/$(whoami)/miners/miner1_env/bin/activate

# 3. Install dependencies (CPU torch version - CRITICAL!)
pip install --upgrade pip
pip install torch==2.7.1 --index-url https://download.pytorch.org/whl/cpu
pip install bittensor==9.7.0
pip install -r requirements.txt

# 4. Set up your wallet
btcli wallet create --wallet.name my_wallet --wallet.hotkey miner1

# 5. Register on subnet 90
btcli subnet register --netuid 90 --wallet.name my_wallet --wallet.hotkey miner1

# 6. Configure and run
export WALLET_NAME=my_wallet
export HOTKEY_NAME=miner1
export API_URL=https://api.subnet90.com
python run_miner.py

$BRAIN Token

What is $BRAIN?

$BRAIN is the native token of Subnet 90, representing stake and governance rights in the DegenBrain verification network. It's a subnet token on the Bittensor network.

The $BRAIN alpha token will earn from the degenpredict.com fees - 30% of the fee will buy $BRAIN allocated to the DAO strategic reserve. More details at https://degenpredict.com/token.

Network: Bittensor
Subnet ID: 90
Type: Subnet Token

How to Buy $BRAIN

  1. Get TAO tokens

    Purchase TAO (Bittensor's main token) from exchanges like Gate.io or MEXC

  2. Set up Bittensor wallet

    Install btcli and create a wallet: btcli wallet create

  3. Access TAOstats

    Visit TAOstats Subnet 90 and connect your wallet

  4. View Subnet 90 Details

    Explore validators, miners, and delegation options for $BRAIN

  5. Stake TAO for $BRAIN

    Delegate your TAO to validators to earn $BRAIN emissions

Note: $BRAIN represents your stake in Subnet 90. By delegating TAO to subnet validators, you earn a share of the subnet's emissions proportional to your stake.
